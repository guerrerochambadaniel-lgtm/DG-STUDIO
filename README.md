# DG STUDIO - Plataforma de Ingeniería Estratégica Digital

Repositorio oficial de la arquitectura web de **DG STUDIO**, agencia especializada en desarrollo web de alto rendimiento, posicionamiento SEO Local y marketing de conversión.

## 🚀 Tecnologías Implementadas (Stack Core)
- **Estructura Semántica:** HTML5 optimizado para accesibilidad y SEO.
- **Estilos y UI:** CSS3 puro con variables nativas, CSS Grid y animaciones de hardware acelerado.
- **Interacción:** Vanilla JavaScript (ES6+), implementando `IntersectionObserver` para scroll-reveal sin dependencias pesadas.
- **Despliegue:** CI/CD automatizado vía Netlify.

## 💼 Líneas de Negocio (Casos de Impacto)
El código base está estructurado para presentar infraestructuras digitales enfocadas en:
1. **Infraestructura B2B:** Digitalización de empresas tradicionales y operativas.
2. **Retail y Contenido:** Estrategias de posicionamiento y multimedia.
3. **Tráfico a Tienda Física:** Geolocalización avanzada en Google Maps.
